# puppy-star-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

# Directory Structure
``` 
├─assets # 资源文件
│  ├─css # css文件
│  └─images # 图片文件
├─components # 共用组件
│  ├─app-main
│  ├─aside 
│  ├─footer 
│  └─header
├─router # 路由文件
├─store # vuex文件
│  └─moduls
└─views # 视图
    ├─home-page # 首页
    │  └─child-comps # 子组件
    └─layout # 最外层路由
        └─chil-comps
```
